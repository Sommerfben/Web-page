Solid State Electric Fence Energizer

A highly efficient electric fence energizer optimized for instant shock delivery and maximized battery life. Based on a two stage approach, this design accepts input voltages from 4.5V to 15V and outputs ~10kV to an electric fence wire.

Key takeaways
  • Designed high voltage circuitry to minimize unit cost
  • Followed PCB layout standards of high voltage circuits to ensure safe operation
  • Carefully selected componentry to balance deterrent performance and safety considerations

Node-LED
IoT Lighting Solution

A fully integrated internet of things lighting solution to allow for custom smart home devices. Its comprised of an ESP8266 and switch mode power supply integrated with 46 individually addressable LEDs to allow for easy deployment. Runs a custom web server to facilitate integration with existing smart home systems. Supports a robust input voltage range to utilize various off the shelf power supplies.

Key takeaways
  • Developed custom web server to allow for wireless control
  • Implemented ASCII character display feature to allow for nuanced communication

Reverse engineering motor control communication protocol

Reverse engineering of the proprietary motor control protocol of the Cinevate camera slider. This was used to implement a custom control interface for increasing repeatability

High Power LED Driver

LED driver capable of driving up to 150W, while featuring current controlled analog dimming. Analog dimming is superior to traditional PWM dimming when driving LEDs for use with cameras as it avoids a strobing effect.

Gamecube controller USB MIDI interface

USB MIDI interface that is fully compatible with a Nintendo Gamecube controller with no hardware modifications to the controller itself. Using a controller port from a Gamecube, this project decodes the proprietary communication protocol used by the controller and converts it into a standard USB HID MIDI interface for use with industry software.

Jumbo LED 7-Seg

Giant 7-segment display characters utilizing a custom mechanical layer stackup to achieve a strong diffusion effect while minimizing light loss. Designed with individually addressable RGB LEDs, the firmware driver was developed to be arduino compatible to ensure easy of use

Key takeaways
  • Created arduino compatible FW driver to support arbitrary number digits
  • Optimized 3D printed interposer layer to maximize light diffusion
